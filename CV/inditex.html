Contexto

A través de Nunegal Consulting, trabajé como Ingeniero de Datos para Inditex, el mayor grupo de distribución de moda del mundo. El proyecto se centró en el desarrollo de software de alta disponibilidad para la gestión y control de calidad (QA) de grandes volúmenes de datos en tiempo real dentro del ecosistema logístico y operativo de Inditex. La escala del dato y la criticidad del sistema exigían un rigor extremo en la calidad y fiabilidad de cada componente.

Contribuciones Clave

🔄

Pipelines de Datos con Apache Kafka

Transformación y procesamiento de datos masivos con Kafka como backbone de mensajería.

  • ▸

    Desarrollo de pipelines de transformación que procesaban flujos de datos de múltiples fuentes del ecosistema Inditex en tiempo real.

  • ▸

    Uso avanzado de Kafka: Streams para transformaciones en vuelo y Exactly Once para garantía de entrega.

  • ▸

    Validaciones de integridad en cada etapa del pipeline, garantizando la consistencia del dato desde el origen hasta el almacenamiento final.

  • ▸

    Gestión de topics, particiones y consumer groups para optimizar el throughput y garantizar la ordenación de eventos críticos.

✅

Quality Assurance de Datos en Tiempo Real

Sistemas de QA automatizado para validar calidad, consistencia y completitud.

  • ▸

    Implementación de validaciones automáticas sobre los datos en tránsito, detectando anomalías, duplicados y datos incompletos en tiempo real.

  • ▸

    Cálculo de métricas de integridad de los datos en tiempo real.

  • ▸

    Definición de reglas de calidad del dato adaptadas a los requisitos de negocio de cada dominio de Inditex.

  • ▸

  • ▸
