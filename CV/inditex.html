Contexto

Trabajé como Ingeniero de Datos para Inditex a través de Nunegal Consulting. Me centré en el desarrollo de software para la validación y el control de calidad (QA) de flujos de datos en tiempo real dentro de sus sistemas de logística y operaciones, asegurando la fiabilidad de la información procesada.

Contribuciones Clave

🔄

Pipelines de Datos con Apache Kafka

Procesamiento de flujos de datos utilizando Kafka.

  • ▸

    Desarrollo de pipelines para procesar flujos de datos de múltiples fuentes de Inditex en tiempo real.

  • ▸

    Uso de Kafka Streams para procesar datos sobre la marcha y configuración Exactly Once para evitar pérdidas o duplicados.

  • ▸

    Validaciones de integridad en cada etapa del flujo para asegurar la consistencia de los datos desde el origen hasta el destino.

  • ▸

    Gestión de topics, particiones y consumer groups para optimizar el rendimiento y asegurar el orden de los eventos.

  • ▸

  • ▸
