# Desarrollador de Software — Vector ITC → Securitas Direct

## 03/2021 — 11/2021 | [Vector ITC](https://www.softtek.com/) → [Securitas Direct](https://www.securitasdirect.es/) | Asturias

---

## Contexto

Como desarrollador en Vector ITC (ahora Softtek), trabajé para Securitas Direct, líder en sistemas de seguridad en Europa. El proyecto se centró en procesado de señales de dispositivos de seguridad físicos, donde la fiabilidad y el tiempo de respuesta eran absolutamente críticos.

---

## Contribuciones Clave

### 🔔 Operadores Automáticos de Señalización

- Desarrollo de operadores que interpretaban señales de dispositivos de seguridad en tiempo real.
- Lógica de decisión automática sobre activación de alarmas y notificaciones a la CRA (Central Receptora de Alarmas).
- Reglas de negocio complejas para reducir falsos positivos sin comprometer la seguridad.

### 📡 Procesado de Señales en Tiempo Real

- Transformación de datos brutos de dispositivos físicos en eventos de seguridad mediante KSQL (Kafka SQL).
- Apache Kafka avanzado: Connect para integración de fuentes, Streams y KSQL para procesamiento en vuelo, administración de brokers y topics.
- Integración con servicios externos mediante clientes REST y SOAP para comunicación con sistemas legacy de seguridad.

### 🚀 CI/CD y Despliegues

- Pipelines CI/CD con Jenkins para testing, build y despliegue en Kubernetes.
- Kubernetes con Helm para gestión de charts de despliegue, y Lens para monitorización visual de clusters.
- Monitorización en Linux: gestión de logs, diagnóstico de servicios y contenedorización con Docker.
