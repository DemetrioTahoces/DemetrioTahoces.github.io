# Ingeniero de Datos — Nunegal Consulting → Inditex
**11/2021 — 09/2022** | La Coruña

## Contexto
Ingeniero de Datos para Inditex (mayor grupo distribución moda mundial) vía Nunegal Consulting. Software alta disponibilidad para gestión y QA de grandes volúmenes datos tiempo real en ecosistema logístico.

## Contribuciones

### Pipelines Kafka
- Transformación flujos datos múltiples fuentes en tiempo real.
- Kafka Streams (transformaciones en vuelo) + Exactly Once (garantía entrega).
- Validaciones integridad en cada etapa del pipeline.
- Gestión topics, particiones, consumer groups para optimizar throughput y ordenación.

### QA Datos Tiempo Real
- Validaciones automáticas datos en tránsito: anomalías, duplicados, datos incompletos.
- Métricas integridad datos tiempo real.
- Reglas calidad adaptadas a requisitos negocio por dominio.

### Monitoreo Grafana
- Dashboards: throughput, latencias, errores, estado consumer groups.
- Alertas proactivas ante degradaciones/inconsistencias.

### Almacenamiento Snowflake
- Consultas analíticas alto rendimiento sobre datos históricos.
- Métricas QA sobre datos almacenados.

## Stack
- Backend: Java 11, Spring Boot, Lombok, API REST, Programación Funcional, SQL.
- Streaming: Apache Kafka (Streams, Consumers, Producers, Exactly Once).
- Monitoreo: Grafana (dashboards + alertas).
- Data Warehouse: Snowflake.
- Infra: OpenShift, GitHub/Azure (CI/CD), JIRA (Scrum).
