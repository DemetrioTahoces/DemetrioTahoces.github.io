# Desarrollador de Software — Vector ITC → Securitas Direct
**03/2021 — 11/2021** | Asturias

## Contexto
Vector ITC (ahora Softtek) para Securitas Direct (líder seguridad Europa). Procesado señales dispositivos seguridad físicos. Fiabilidad y tiempo respuesta críticos.

## Contribuciones

### Operadores Automáticos Señalización
- Interpretación señales dispositivos seguridad en tiempo real.
- Lógica decisión automática: activación alarmas y notificaciones CRA (Central Receptora Alarmas).
- Reglas negocio para reducir falsos positivos sin comprometer seguridad.

### Procesado Señales Tiempo Real
- KSQL (Kafka SQL) para transformar datos brutos en eventos seguridad.
- Kafka avanzado: Connect (integración fuentes), Streams, KSQL, administración brokers/topics.
- Clientes REST y SOAP para comunicación con sistemas legacy.

### CI/CD y Despliegues
- Jenkins: pipelines testing, build, despliegue en Kubernetes.
- Kubernetes con Helm (charts) y Lens (monitorización clusters).
- Monitorización Linux: logs, diagnóstico servicios, Docker.

## Stack
- Backend: Java, Spring Boot (Spring Kafka), Lombok, MapStruct, clientes REST/SOAP, Git.
- Streaming: Kafka (Connect, Streams, KSQL, administración brokers).
- CI/CD: Jenkins, JIRA (Scrum).
- Infra: Kubernetes (Helm, Lens), Docker, Linux.
