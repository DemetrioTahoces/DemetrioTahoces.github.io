# Ingeniero de Software & Especialista en Robótica e IA — Alisys S.L.U
**09/2019 — 03/2021** | alisys.net | Asturias

## Contexto
Alisys: soluciones digitales. Evolución en 1,5 años de Ingeniero de Software (Android) a Especialista en Robótica e IA (adaptación rápida a dominios complejos).

## Fase 1 — Ingeniero de Software (09/2019 — 01/2020)
### App Android Mensajería
- Mensajería multicanal con adjuntos multimedia (imágenes, vídeos, documentos, audios).
- Push con Firebase Cloud Messaging (FCM) + autenticación Firebase.
- Backend CloudCX para BBDD y blob storage.
- Sincronización offline con resolución conflictos al recuperar conectividad.

## Fase 2 — Especialista en Robótica e IA (01/2020 — 03/2021)
### API REST Plataforma Robótica
- API REST Java como orquestador central: robots Pepper, NAO, James, Temi. Backend VoIP Asterisk.
- WebSockets Java: monitorización y control estado componentes en tiempo real.
- Kafka: productores/consumidores para datos sensorización robots.

### Microservicios IA
- Reconocimiento Facial: identificación/verificación usuarios (Android nativo).
- STT: Google Cloud Speech, Azure Cognitive Services.
- NLU: Google Assistant, Dialogflow, Rasa.

### Apps Android para Robots
- Apps embebidas en robots para VoIP y WebRTC.
- Streaming vídeo y videollamada con WebRTC, señalización SIP con Asterisk.

## Stack
- Backend: Java (API REST, WebSockets), Spring Boot, Kafka (sensores).
- Mobile: Android (Java/Kotlin), Firebase (FCM, Auth), Reconocimiento Facial nativo.
- IA: STT (Google Cloud, Azure), NLU (Google Assistant, Dialogflow, Rasa).
- Comunicaciones: VoIP (Asterisk), WebRTC, SIP, GitLab.
